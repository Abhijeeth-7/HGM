How to use the generated code

  1. Click "Generate Code" button in this site and Copy the code

  2. Open notepad and paste the copied code

  3. Save the notepad file by clicking save as option instead of save

  4. while saving the file add ".py" at the end of your file name

    Ex: If your naming the file as "testCode1" then add it as "testCode1.py"

  5. Set "Save as" to "All files" and Click save

  6. Run the code, by double clicking on the saved file

How to use the configured hand gestures

The gestures are isolated in 5 different modes of operations

Modes

  • Mode 1 : All mouse functionalities, use mode 1 when you want to control the mouse

  • Mode 2 : Contains all Swipe Gestures

  • Mode 3 : Contains all Static Gestures

  • Mode 4 : Contains all Action Gestures

  • Mode 5 : Can contain action gestures, swipe gestures or both depending on your configuration.

How to Switch Modes

You can switch between the modes by brining both the hands into webcam view port and specifying the mode using your fingers on the left hand as shown

How to Operate the mouse in mode 1

  1. To operate mouse change your hand state to the following state -> Open Thumb, Index, and Little fingers and close remaining fingers
  2. To perform "left click" toggle your "index finger" I.e, close and open your index finger once
  3. To perform "right click" toggle your "little finger" I.e, close and open your little finger once
  4. To avoid detection or moving mouse pointer make a Fist!

Tips

  • To Avoid detection at any point of time, make a Fist!

    when all fingers are closed, this state of hand is being used to avoid detection at any point of time. So You can avoid triggering any unintended actions by making a fist

  • The gestures in Mode 1 and 5 can be used on any application.

  • The gestures in Mode 2, 3, and 4 are application specific.
